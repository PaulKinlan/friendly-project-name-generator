Friendly Project Name Generator

A simple library to generate random project names (much like Glitch), using the friendly-words project (by Glitch)

If you want to generate random project names (in the form of "X-Y") make a web request to https://friendly-project-name.kinlan.now.sh/api/names



    
You can control how many names to generate with the a query-string parameter of count

    


    
You can control separator with the a query-string parameter of separator. i.e, separator=@
